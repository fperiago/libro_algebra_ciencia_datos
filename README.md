# libro_algebra_ciencia_datos
Este repositorio contiene material complementario al libro con título Algebra Lineal para Ciencia de Datos cuyos autores son Claudi Busqué (Universidad de Murcia) y Francisco Periago (Universidad Politécnica de Cartagena).
En concreto, cada capítulo de libro incluye notebooks de Jupyter para que el alumno pueda afianzar sus conocimientos de Álgebra a través de Python.
